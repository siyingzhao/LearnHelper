# Learn Helper

本项目基于[Harry-Chen/Learn-Helper: 清华大学网络学堂助手](https://github.com/Harry-Chen/Learn-Helper)开发，主要增加了一些学期作业情况统计和可视化功能，目前支持Chrome/Edge/Firefox浏览器插件功能，新增功能主要包括以下内容：

- **提交日历热力图**
- **提交时间热力图（周-小时）**
- **提交时间线**
- **提交星期分布**
- **提交延时图**
- **未提交作业风险分布**
- **提交时段分布（小时）**
- **附件大小分布**
- **个人拖延画像**
- **最近提交**

期末周仓促摸鱼并在AI辅助下完成，纰漏甚多，欢迎大家多提issue lol
